# Daily Schedule (every day)

The fixed skeleton. Weightlifting locked at 10:00. Project work fills the 💻 blocks —
which specific SSA task goes in each block comes from that day's all-day task banner
(see [[RESEARCH_AND_OUTREACH_WEEK]] and the `Tasks/` folder).

| Time | Block | |
|------|-------|---|
| 06:30 – 06:45 | 🌅 Wake up + hydrate | |
| 06:45 – 07:15 | 🚿 Morning routine | wash, dress |
| 07:15 – 07:45 | 🍳 Breakfast | |
| **07:45 – 10:00** | 💻 **Deep Work 1 — SSA** | hardest task, peak focus |
| **10:00 – 11:00** | 🏋️ **Weightlifting** | non-negotiable |
| 11:00 – 11:30 | 🚿 Shower | |
| 11:30 – 12:15 | 🥗 Lunch | off-screen |
| **12:15 – 15:00** | 💻 **Deep Work 2 — SSA** | main build/research |
| 15:00 – 15:20 | 🚶 Break | walk / stretch |
| **15:20 – 17:30** | 💻 **Deep Work 3 — outreach** | emails, follow-ups |
| 17:30 – 18:30 | 🍽️ Dinner | fully unplug |
| 18:30 – 20:00 | 📚 Learning | orbital mechanics / reading |
| 20:00 – 21:30 | 🌙 Wind down / personal | |
| 21:30 – 22:00 | ✅ Review + plan tomorrow | tick tasks, set top 3 |
| 22:00 | 😴 Lights out | 8+ hrs |

**Productive total:** ~6.5 hrs deep work + 1.5 hrs learning ≈ **8 hrs/day** (~50 hrs/week, on target).

These live on the TaskNotes calendar as **recurring daily tasks** (`Routine/` folder) —
tick each one as you go. Adjust any time in the calendar UI and TaskNotes rewrites it.
