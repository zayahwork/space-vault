# 📬 Guide — Handling Replies & Follow-ups

## When ANY reply lands

1. **Answer within the hour if possible.** Speed reads as respect and seriousness.
2. Goal of every exchange: **get a 15–20 minute call.** Offer 2–3 concrete times.
3. On the call: LISTEN. You're hunting for them describing pain *unprompted*. Ask:
   - "Where does tracking/conjunction stuff actually hurt you day-to-day?"
   - "What do the current tools/free feeds *not* do?"
   - "If that got fixed, what would it be worth?" (only if it flows naturally)
4. After: log it below + tell Claude → we update the scoreboard and mine it for the wedge.

## Nudges (for the silent ones)

- Wait **3+ business days**, then ONE line: *"Bumping this in case it got buried — no worries if now's a bad time."*
- No response after a nudge = let them go, gracefully. Never a second nudge.
