Do this once the list above it is mostly ticked (roughly the weekend). With Claude, write
`03 Reference/Week 1 Findings.md` answering honestly:

  is a superpower, not a failure.)

## Then decide week 2's shape
- **Strong signal** (real conversations, a "can I try it?") → double down: build well, book more
  calls, aim the [[Guide - Wedge Memo|Wedge Memo]] at the next SBIR window
- **Weak signal** → more outreach first, try a different segment (insurers? missed operators?),
  keep building lighter
- **26.BX had a perfect topic** → decide the Aug-18-sprint question with eyes open

## The scoreboard verdict
🗣️ 5+ unprompted pain descriptions? 💰 1+ "can I try it?" — If yes to both, the week worked.
If no: that's data too, and the plan says exactly what to do with it.
